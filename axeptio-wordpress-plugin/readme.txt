Axeptio allows you to make your website compliant with GDPR.

== Description ==
The Axeptio Plugin downloads our SDK and allows you to enter your project ID in the settings to display the module on your website.

== Installation ==
1 - Download the plugin and install it on your Wordpress website
2 - Login to our back office and configure your widget https://admin.axeptio.eu
3 - Input your project ID in the Wordpress admin table and save. Your version will load : select it from the list and save.
4 - Activate the widget, it will be displayed on the frontend. Use a Tag Manager to handle which cookies to submit to consent.
